# 🌟 Aula de Web Design – Atividade 01

Este projeto consiste em uma **página web interativa desenvolvida para fins educacionais**, como parte da **Atividade 01 da aula de Web Design**.  
O objetivo é demonstrar, na prática, a integração entre **HTML, CSS e JavaScript**, explorando conceitos de estrutura, estilização, animações e interatividade.

---

## 🎯 Objetivo da Atividade

- Aplicar conceitos fundamentais de **desenvolvimento web**
- Integrar **HTML, CSS e JavaScript** em um único projeto
- Trabalhar interatividade com o usuário
- Explorar recursos visuais modernos (animações e efeitos)

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura da página  
- **CSS3** – Estilização, animações, layout responsivo e efeitos visuais  
- **JavaScript** – Interatividade, manipulação do DOM e lógica da aplicação  

---

## 📌 Funcionalidades Implementadas

- Modal de boas-vindas com entrada de nome e idade
- Saudação personalizada ao usuário
- Exibição dinâmica do dia da semana
- Seção explicativa sobre HTML, CSS e JavaScript
- Botão interativo “Saiba Mais”
- Cards com efeito *flip* explicando tecnologias
- Laboratório interativo de JavaScript:
  - Alteração dinâmica de texto
  - Alteração dinâmica de cores
- Animação de fundo simulando “chuva de código”
- Layout responsivo para diferentes tamanhos de tela

---

## 🎨 Conceitos Trabalhados

- Manipulação do DOM
- Eventos em JavaScript
- Condicionais e funções
- Animações com `@keyframes`
- Glassmorphism (efeito de vidro)
- Responsividade com *media queries*
- Boas práticas de organização visual
